毕业了，完结撒花

毕业感想

一眨眼就毕业了，内心感慨万千，总的来说还是不错的，收获了很多朋友，挺铁的朋友，这是一种无形的财富吧（狗富贵，勿相汪）。说到我自己，大学学到的东西其实不是很多，原因嘛。。。。 大多都是自学的比如iOS、PHP 但是学的都不怎么样，也就是也表面的东西，懂得如何使用三方库，会用API。。对于技术深层次的理解还差的很多，大概就是这样吧。毕业了，也就是面临工作了，之前一直在找 iOS 的工作，但是基于目前的行业行情来说，我这种小白真心不好找，所以呢。。还是先静下心来好好学习一波吧。大学嘛，对于学校本身没有太多可以留恋的，值得留恋的还都是那群人。。。。

学习计划

先好好学习一波吧。多多了解深入的知识。最近买了好多的书还有以前买的书没有看，给自己整理一下要看的书，以及学习计划吧。

  • Effective Objective-C
    很多大佬都推荐这本书，我也初步的看了一下，就一下子被吸引了，这本书讲到很多底层的知识和一些很有用的技巧等。。。非常值得深入阅读，把其中的知识点都学会了，对于 iOS 开发有很大的帮助。

  • Objective-C 高级编程 iOS 与 OS X 多线程与内存管理
    这本书对于GCD 的使用和内存管理都做了详细的解释，还有对于 block 的原理有很详细的介绍，很值得阅读。

  • Swifter 开发者必备 Tips
    喵神写的书，对于学习 Swift 有极大帮助。很早以前买的，可惜一直没有时间读。

  • Swift 面向协议编程
    对于面向协议有些了解吧。还没怎么看。

  • RxSwift Reactive Programming With Swift
    在之前的毕设中也尝试了一下响应式编程，感觉还是很不错的。 在看了 bestswifter 大神的推荐，感觉还是挺不错的，对于新手的我来说，是个很不错的学习，但是价格对于现在的我来说还是有点贵的，等我工作了再买吧。~~~~

暂时就这么多吧， 大学就结束了，曾不止一次感叹4年在眨眼间过去，4年了，我的学生生涯也就此结束，感慨良多。。。。
